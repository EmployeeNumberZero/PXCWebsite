Frequently Asked Questions

  • For NPN drugs please add name in notes. NPN auto population will be available next update.

  • All posted ads are attached to a pharmacy not pharmacist! Please remember that when deleting/editing a pharmacy from your profile. All attached listings will be deleted automatically and messages archived.

  • Contact the posting ad via phone or through our messaging app. Verify the contact information via google etc. if needed.

  • Report any suspicious listings to our email contact@pharmacyxchange.ca - add as much information as possible and screenshots.

  • Report any technical difficulties using contact@pharmacyxchange.ca

  • For license numbers under 8-9 digits please add zeros to the beginning. For example (000001234) instead of 1234.

  • Verify the other pharmacist's licence! For example in Ontario use https://ocpinfo.com/

  • Never disclose patient information to any user on the application. We will never ask for it.

  • You can only see postings related to the province the user registered the profile with.

  • If you have more than one license in different provinces please set up a separate account for it.

  • For advertising opportunities please contact us at contact@pharmacyxchange.ca
